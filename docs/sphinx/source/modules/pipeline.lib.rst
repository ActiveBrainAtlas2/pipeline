Library modules
---------------

These files contain classes and methods used in peforming actual maninpulations
on the numpy arrays (the images). This involves:

- CZI extraction
- Metadata extraction
- Histogram creation
- Mask creation
- Image cleaning
- Image alignment
- Neuroglancer precomputed data creation
